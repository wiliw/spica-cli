# Spica Project Config

## Project Info
- Type: TypeScript CLI
- Framework: Node.js + Ink
- Language: TypeScript

## Commands
- Build: `npm run build`
- Test: `npm test`
- Dev: `npm run dev`

## Constraints
- Code style: No comments unless asked
- Testing: Vitest framework

## TUI Architecture
- Split layout: 50% left + 50% right (66% thinking + 33% tools)
- Custom scroll via useScroll hook
- Marquee animation for overflow content
- Input isolation with React.memo
