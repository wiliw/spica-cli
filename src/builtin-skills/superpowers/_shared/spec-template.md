## Template Usage Notes

**Required sections:**
- Summary (one sentence)
- Requirements (checkbox format, specific and verifiable)
- Architecture (2-3 sentences minimum)
- Testing Strategy

**Optional sections:**
- Components (if multi-component)
- Data Flow (if non-trivial)
- Constraints (if applicable)
- Open Questions (if unresolved)

**Status values:**
- Draft: Still being refined with user
- Approved: User approved, ready for planning
- Implemented: Plan executed, work complete

**Plan link:** Fill in after writing-plans skill creates the plan file.
